# 饮食规划 Web 应用 PRD

## 1. 项目概述

- **项目名称**: 饮食规划助手
- **项目类型**: Web 应用 (Next.js + Tailwind CSS + Cloudflare Pages)
- **部署地址**: https://wizmealplanner.pages.dev/
- **GitHub**: https://github.com/wiz90/wizmealplanner

## 2. 功能模块

### 2.1 档案设置 (page=1)
用户设置个人饮食偏好

| 功能 | 说明 |
|------|------|
| **目标选择** | 减脂/增肌/制作简单/便当/替代外卖/清淡 |
| **忌口选择** | 海鲜/牛肉/猪肉/鸡肉/奶制品/坚果/麸质/酒精/生冷/辣 |
| **不爱吃** | 预设15种 + 自定义，区分"一点不吃"/"少吃" |
| **厨具选择** | 炒锅/平底锅/电饭锅/空气炸锅/微波炉/烤箱/电压力锅/汤锅/蒸锅 |

### 2.2 规划设置 (page=2)
设置生成参数

| 功能 | 说明 |
|------|------|
| **风格选择** | 共16种 (家常菜/川菜/湘菜/粤菜/闽菜/东北菜/清真/日式/韩式/泰式/越南/意式/西式简餐/轻食沙拉/Wagas风格/素食) |
| **天数** | 1-30天，默认3天 |
| **人数** | 1-20人，默认2人 |
| **餐次** | 早餐/上午茶/午餐/下午茶/晚餐，默认3餐 |
| **预算** | 无限制 |
| **制作简单开关** | 减少备菜流程 |

### 2.3 推荐结果 (page=3)
AI 生成的菜单展示

| 功能 | 说明 |
|------|------|
| **日期选择** | tab切换查看不同天 |
| **菜品卡片** | 可展开做法 |
| **换一道** | 替换单个菜品 |
| **换一批** | 重新生成所有 |
| **备菜清单** | 跳转备菜页 |

### 2.4 备菜清单 (page=4)
生成购物清单

| 功能 | 说明 |
|------|------|
| **食材汇总** | 去重合并相同食材 |
| **勾选功能** | 标记已购买 |
| **分量计算** | 多人份自动计算 |

### 2.5 历史记录 (page=5)
查看历史记录

| 功能 | 说明 |
|------|------|
| **记录列表** | 按时间倒序 |
| **查看详情** | 点击查看完整菜单 |
| **删除单条** | 删除指定记录 |
| **清空全部** | 清空所有记录 |

### 2.6 欢迎页 (page=0)
新用户/返回用户首页

| 功能 | 说明 |
|------|------|
| **新用户** | 显示"填写档案"按钮 |
| **有档案用户** | 显示"开始规划"/"修改档案" |
| **快捷入口** | "历史记录"按钮 |

## 4. API 设计

### 4.1 生成接口
`POST /api/generate`

```json
{
  "profile": {
    "goals": ["健康"],
    "hard_restrictions": ["海鲜"],
    "custom_restrictions": [],
    "kitchen_tools": ["炒锅"]
  },
  "session": {
    "style_preferences": ["家常菜"],
    "soft_dislikes": ["香菜"],
    "days": 3,
    "meals_per_day": ["早餐", "午餐", "晚餐"],
    "person_count": 2,
    "budget": "无限制",
    "simple_prep": false
  }
}
```

### 4.2 返回格式
```json
{
  "days": [
    {
      "day_index": 1,
      "meals": [
        {
          "type": "午餐",
          "dishes": [
            {
              "dish_type": "主食",
              "recipe": {
                "recipe_name": "米饭",
                "time_cost": 30,
                "ingredients": ["大米100g", "水"],
                "steps": ["淘米", "加水", "煮熟"]
              }
            }
          ]
        }
      ]
    }
  ]
}
```

## 5. 数据存储

- **方案**: localStorage
- **keys**:
  - `meal_profile`: 用户档案
  - `meal_history`: 生成历史 (最多10条)
  - `meal_last_result`: 上次结果 (用于"换一批")
