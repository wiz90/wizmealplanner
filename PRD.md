# 饮食规划助手 - PRD

> 更新日期: 2026-03-10

---

## 一、完整用户流程

```
欢迎页
├── 新建规划 → 档案页 → 计划页 → 生成结果 → 备菜页
├── 修改档案 → 档案页
└── 历史记录
```

---

## 二、已完成 ✅

### 1. 欢迎页
- 档案预览卡片 (标题 + 修改按钮)
- 有/无档案不同入口
- 历史记录入口

### 2. 档案页
- 目标: 多选 (制作简单)
- 忌口: 预设 + DIY 自定义
- 不爱吃: 预设 + DIY + 程度 (一点不吃/少吃/适量)
- 厨具: 多选
- localStorage 持久化

### 3. 计划页
- 风格: 13 种 (家常/地中海/轻食/日式/韩式/东南亚/高蛋白/川味/粤菜/素食/西式/创意菜/沙拉轻食)
- 天数: 1/3/5/7 + 自定义 (1-30天)
- 餐次: 5 种 (早餐/上午茶/午餐/下午茶/晚餐)

### 4. 生成结果页
- 顶部显示本次计划 (风格 + 天数 + 目标)
- 天数 Tab 切换
- 菜品类型标签 (主食/主菜/蔬菜)
- 做法可展开/收起
- 当天总耗时
- "换一批" / "备菜清单" 按钮

### 5. 备菜页
- 食材按分类 (肉类/蔬菜/调料/其他)
- 勾选功能
- 勾选计数

### 6. 历史记录页
- 显示最近 10 条记录

### 7. 生成功能
- DeepSeek API
- 报菜名 Loading 动画
- 生成后自动保存历史
